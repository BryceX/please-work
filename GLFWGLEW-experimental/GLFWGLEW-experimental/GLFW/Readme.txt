 This is a small game engine in C++ using the OpenGL framework capable of animating sprites, displaying text, and accepting key input to move them across the screen.

Text
Draw(float xPosition, float yPosition, char);
Takes and X and Y position followed by letters to draw a letter to screen.

Draw(float xPosition, float yPosition, string);
Takes and X and Y position followed by letters to draw words to screen.

Move();
WASD to move Up, left, down, right

Asteroids
Draw();
Draws a green hexagon in a random location.

Stars
Draw();
Draws a white pixel in a random location.

Player
Draw(texture.png);
Draws a triangle with a texture.
